04/01/2017

EBE has been updated. Version 219, and with it a lot of cool new features, blocks, smileys, and of course commands! But that's not all. We also earned a bigger community! New players have been joined to our little game as well, And EBE is starting to grow! So lets list all the new stuff we have added here! The first thing is, /help list is now sorted alpha-betic.

New Commands!

  • -giveeffect (/geffect [player] [effect name / effect ID]), Owner-Only, Need the effect to give it to a person.
  • -removeeffect (/reffect [player] [effect name / effect ID), Owner-Only.
  • -setteam (/setteam [player] [team colour / team ID]), Owner-Only, Need the teams to give it to a person.
  • -givecrown (/givecrown [player]), Owner-Only.
  • -removecrown (/removecrown), Owner-Only.
  • -geffectall (/giveeffectall [effect name / effect ID]), Owner-Only, Need the effect to give it to a person.
  • -setteamall (/setteamall [team colour / team ID]), Owner-Only, Need the teams to give it to a person.
  • -reffectall (/removeeffectall [effect name / effect ID]), Owner-Only.
